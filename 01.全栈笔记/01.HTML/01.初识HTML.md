其实这个内容用一个笔记就可以讲完，因为 HTML 根本就不是编程语言，她只是一个标签表达语言而已，非常的简单。
我们在浏览器或者说一些应用里看到的界面，有文字，有图片，等等乱七八糟的东西其实都是 HTML+CSS+JS 合并起来的产物。

## HTML
[HTML-MDN](https://developer.mozilla.org/zh-CN/docs/Web/HTML)
超文本标记语言，都说了是标记了，所以他就是起到一个标记作用，把内容放里面，让浏览器知道哪块是什么内容，然后渲染出来罢了，比如标题，段落，链接，图片，文件等等。

``` html
<h1>Heading</h1>
```
比如上面这个就是告诉浏览器，我这个标签包裹起来的内容是一个一级标题，浏览器需要以一级标题的样式把他渲染出来。

可以看到，对于一般的内容，我们需要像肉夹馍一样，把他拿两个标签包裹起来。
当然很明显的其实就是一个标签，一个表示开始，一个表示结束而已，比如我们上面的 h1 标签。结束的话就在前面加上一个/就可以了，就是这么简单。

那么我们现在来正式的学习一下标签吧，不过说实话 HTML 的标签是有很多的，但是其实我们平时经常用的也就是那么 10 几个。不用有太多的压力，想不起来的时候去差哪个内容用什么标签来表示比较好就可以了。

## Heading -标题 标签
就是我们拿来举例的那个标签，也就是用来表示我们的标题的标签。上面我们的 h 1 就是一级标题标签，html 一共有 1 到 6 6 个层级标签，从视觉上来看一级标题显示最大，六级标题显示最小。
``` html
<h1>Hello, World!</h1>
<h2>Hello, World!</h2>
<h3>Hello, World!</h3>
<h4>Hello, World!</h4>
<h5>Hello, World!</h5>
<h6>Hello, World!</h6>
```

## Paragraph-段落标签
这个时候就会有人问了，我直接打字在上面我看也是可以正常显示的，为啥非得用段落标签。
例子
``` text
你好
很高兴认识你
```
你把这个输入，你的本意是你想换行，但其实你会发现他被渲染在一行上了，仅仅是中间有个小空格而已。

所以我们要用 p 标签进行包裹
``` html
<p>你好</p>
<p>很高兴认识你</p>
```
这样你会发现他正常换行了

## 自闭标签
上面我们学了 h 和 p 标签，我们知道这两个都是得一前一后有开始和结束标签的，但其实有的元素我们不需要有任何内容，比如一个水平分割线或者我只是想在某个地方强行换行。
### Horizontal Rule
也就是水平分割线，前面学习的两个标签我们就能够看出来其实标签大部分是以首字母来进行命名的，所以这个的标签就是 hr

但是我们不需要输入任何内容，我们只是想加入一个水平分割线而已，所以我们只需要按照自闭标签的形式输入一个就可以，不需要什么结束标签

也就是标签名+空格 +/

例子
``` html
<p>你好</p>
<hr />
<p>很高兴认识你</p>
```
当然你进入 vscode 输入 hr 你会发现不需要这个/ 因为新的标准已经不需要了，直接写 hr 即可，但是为了兼容性还是建议加上的，都是可以的。

### Break 换行标签
这个就是能在段落之间强行进行换行。
```
<p>春眠不觉晓，处处闻啼鸟。夜来风雨声，花落知多少。</p>
```
我们就可以用这个来换行
```html
<p>春眠不觉晓，<br />处处闻啼鸟。<br />夜来风雨声，<br />花落知多少。</p>
```
我的建议是不要乱用，看情况选择合适的方式进行表达。

至此我们所学习的标签已经可以完成一个简单的网页排版了
``` html
<h1>春晓</h1>
<p>李白</p>
<p>春眠不觉晓，<br />处处闻啼鸟。<br />夜来风雨声，<br />花落知多少。</p>
```
