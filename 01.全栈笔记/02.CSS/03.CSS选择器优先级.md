## 选择器优先级
首先我们要回顾以及总结一下 css 都可以通过哪些方式添加
### 行内样式
```html
<p style="color: blue">测试文本</p>
```

### 内部样式
```html
<!doctype html>
<html lang="en">
  <head>
    <meta charset="UTF-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <title>Document</title>
    <style>
      .test {
        color: blue;
      }
    </style>
  </head>
  <body>
    <p class="test">测试文本</p>
  </body>
</html>

```

### 外部样式
```html
<!doctype html>
<html lang="en">
  <head>
    <meta charset="UTF-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <title>Document</title>
    <link rel="stylesheet" href="./style.css" />
  </head>
  <body>
    <p class="test">测试文本</p>
  </body>
</html>
```

```css
.test {
  color: blue;
}
```

其中优先级行内样式 > 内部样式 > 外部样式。

对于外部样式我们有
#### ID 选择器
```html
<!doctype html>
<html lang="en">
  <head>
    <meta charset="UTF-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <title>Document</title>
    <link rel="stylesheet" href="./style.css" />
  </head>
  <body>
    <p id="test">测试文本</p>
  </body>
</html>
```

```css
#test {
  color: blue;
}
```

#### 类选择器/属性选择器
##### 类选择器
```html
<!doctype html>
<html lang="en">
  <head>
    <meta charset="UTF-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <title>Document</title>
    <link rel="stylesheet" href="./style.css" />
  </head>
  <body>
    <p class="test">测试文本</p>
  </body>
</html>
```

```css
.test {
  color: blue;
}
```

##### 属性选择器
```html
<!doctype html>
<html lang="en">
  <head>
    <meta charset="UTF-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <title>Document</title>
    <link rel="stylesheet" href="./style.css" />
  </head>
  <body>
    <p data-test="active">测试文本</p>
  </body>
</html>
```

```css
[data-test] {
  color: blue;
}
```

##### 元素选择器
```html
<!doctype html>
<html lang="en">
  <head>
    <meta charset="UTF-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <title>Document</title>
    <link rel="stylesheet" href="./style.css" />
  </head>
  <body>
    <p>测试文本</p>
  </body>
</html>
```

```css
p {
  color: blue;
}
```

这里的权重就是
ID 选择器 > 属性选择器 > 类选择器 > 元素选择器

> [!TIP]
> 尽管很多教程说是属性选择器和类选择器权重相同，不过就调查来看其实还是属性选择器权重高。

## 强制覆盖 important

其实不是很建议用这个，这玩意容易被滥用造成屎山 bug。一般是用来强制的修改某些别站的样式用来实现第三方主题的时候才用，优先级最高。比如永硕 E 盘以及博客园等网站。

```html
<!doctype html>
<html lang="en">
  <head>
    <meta charset="UTF-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <title>Document</title>
    <link rel="stylesheet" href="./style.css" />
  </head>
  <body>
    <p>测试文本</p>
  </body>
</html>
```

```css
p {
  color: blue;
}

p {
  color: red !important;
}
```

反正我的建议是尽量使用 class，避免屎山代码。

其实最后一个代码还涉及到了一点，就是 css 的次序问题，和代码的执行次序一样，css 也是从上往下执行，所以其实最后一个即使不加 important，他还是会被覆盖，这里只是为了演示所以简化了。写的时候一定要注意次序问题。
