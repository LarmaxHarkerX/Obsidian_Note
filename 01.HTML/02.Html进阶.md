## List 列表
在 html 中我们的列表就两个，有序列表和无序列表，虽然是这么叫，但其实还是根据我们写的顺序来呈现列表的。区别就是列表项目前面有没有编号罢了。
我们需要掌握 ordered list unordered list 以及 list element 的用法

其实很简单，就是先写上你要写的是无需表还是有序表，在写上列表项就可以了

``` html
<ul>
  <li>1</li>
  <li>2</li>
  <li>3</li>
  <li>4</li>
  <li>5</li>
</ul>
<ol>
  <li>1</li>
  <li>2</li>
  <li>3</li>
  <li>4</li>
  <li>5</li>
</ol>
```

![image.png](https://raw.githubusercontent.com/LarmaxHarkerX/Obsidian_Images/main/image/20260114002714504.png)

然后我们需要注意的是，html 不那么死板，可以结合出很多的东西，比如无序表嵌套有序表等等

``` html
<ul>
  <li>
    1
    <ol>
      <li>1</li>
      <li>2</li>
      <li>3</li>
      <li>4</li>
      <li>5</li>
    </ol>
  </li>
  <li>2</li>
  <li>3</li>
  <li>4</li>
  <li>5</li>
</ul>
```
![image.png](https://raw.githubusercontent.com/LarmaxHarkerX/Obsidian_Images/main/image/20260114002926904.png)

至此我们列表也就学完了，其实没啥难的，就是这么直白。

## Anchor锚元素与标签属性
这个算是引入标签和属性的最好例子了

简单来讲 html 的标签是有一些属性可以写进去的，比如能否拖拽等等。

锚点元素就是我们平时点击一个页面上的文字，他会跳转到另外一个网页，这就是锚元素。
但是假设我们只是写上比如
```html
<a>GITHUB</a>
```
他是百分之百没法带你打开 github 的，因为你没写上链接, 你会发现他就像是一个文本一样，点击也没反应。

用 vscode 的会发现输入 a 并回车，他会自动给你加上一个 href 属性
```html
<a href=""></a>
```
Mdn 给的解释是
> href
> 超链接所指向的 URL。链接不限于基于 HTTP 的 URL——它们可以使用浏览器支持的任何 URL 协议：
> 
> 使用文档片段链接到页面的某一段
> 使用文本片段链接到某一段文字
> 使用媒体片段链接到某个媒体文件
> 使用 tel: URL 链接到一个电话号码
> 使用 mailto: URL 链接到一个邮箱地址
> 如果 web 浏览器不能支持其他 URL 协议，网站可以使用 registerProtocolHandler()

其实看这么多也没用，你就记得他能加链接就可以了。
```html
<a href="https://github.com">github</a>
```
这个时候你会发现这个文字变蓝了，可以点击了，点击就进入到了 github 首页。

当然我们的属性有很多，比如有的时候我们希望我们页面的东西是可以被拽动的。

```html
<p draggable="true">拖拽测试</p>
```

这个其实用处很大，经常删库的小伙伴可以直接选择库名然后直接拖拽到输入框。用 vscode 的同学可以直接选择要移动的内容拖拽到需要放置的地方。

``` html
<h1>my favourite websites</h1>
<ul>
  <li><a href="https://github.com">github</a></li>
  <li><a href="https://youtube.com">youtube</a></li>
  <li><a href="https://google.com">google</a></li>
</ul>
```

## 网页头部信息
我是喜欢这么叫，用 vscode 的朋友只需要建立 html 后在页面输入！并回车即可得到如下内容。
```html
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Document</title>
</head>
<body>
  
</body>
</html>
```

可以看到他第一步就声明了这个文档的类型是 html，我们在这里可以设置网页的语言，编码，标题，我们也可以在这里引入我们后面要学的 html 和 css 文件。然后把我们的内容写到 body 里面。基本就是这样。
