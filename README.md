## LAERMAX 的笔记仓库
最后的最后，还是换回 obsidian 了 😂，怪尴尬的，因为我发现其实搭配 piclist 加私有仓库的话，体验和本地数据库是一样的。

AFFINE 不是说不好用，主要是稳定性太差，动不动就蓝屏，内容一多就容易出现错误。当然蓝屏问题其实貌似是 Windows 11 的问题，如果你的机器 cpu 不是分大小核心的话千万不要装!!!

我倒是挺喜欢 Affine 的画板的，插入很方便，就是加入尺子工具以及优化线条工具就更好了。

用 Windows 11 的日子还是挺煎熬的，感觉机器跟有脾气一样动不动就蓝屏，还容易损坏硬盘文件。写个文章都提心吊胆。我觉得微软把测试团队辞退转为 Windows insider（就是不用团队测试，做了就往 DEV 频道发，用户发反馈） 是一个十分错误的选择。

我现在就是用 windows 10 iot ltsc 2021 养老，这用了几天了没出现问题，就是肯定是缺内容，但不影响。装上的那一刻心灵都净化了，没有复杂的 ui，反应迅速。

[下载地址](https://massgrave.dev/windows_ltsc_links)

总之就先用着吧。

其实之前放弃 Obsidian 的原因就是只要图片一多，就会在一个地方满地拉屎。不过这次重新搞，配合 piclist 的话，就不会在我硬盘里拉屎了，转而在 gihub 仓库拉屎了（。
